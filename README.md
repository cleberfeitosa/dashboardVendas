# dashboardVendas
 Apliquei as Heurísticas de Nielsen (especialmente Visibilidade do Status do Sistema, Prevenção de Erros e Correspondência com o Mundo Real) para criar uma interface intuitiva. Usei uma paleta de cores moderna (Violeta/Fúcsia) que remete a um ambiente dinâmico de vendas, destacando a vencedora com clareza.
